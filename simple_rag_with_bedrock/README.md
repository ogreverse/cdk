# simple RAG with Bedrock

## Concept

Bedrock を使った簡易な RAG 環境の構築

## Commands

- `npm run build`
  - TypeScript を JS にコンパイル
- `npm run watch`
  - 変更を監視しコンパイル
- `npm run test`
  - Jest ユニットテストを実行
- `npx cdk deploy`
  - このスタックをデフォルトの AWS アカウント/リージョンにデプロイ
- `npx cdk diff`
  - デプロイされたスタックと現在の状態を比較
- `npx cdk synth`
  - 合成された CloudFormation テンプレートを出力
